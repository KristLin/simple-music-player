# Intro:
- A simple electron based music player.
- This project is just for learning.

# Technology Stakc:
- **Node.js**
- **NPM**
- **Bootstrap**
- **Electron**
- **Electron Builder**


# To Use
```bash
# Install dependencies
npm install
# Run the app
npm start
```

# To Build
```bash
# Install dependencies
npm install
# Run the electron builder
# Optional: set icon by adding "--icon=icon.ico"
npm run dist
```
